## 📝 O que é este projeto?

Convenhamos: ninguém merece abrir um perfil no GitHub e dar de cara com 50 arquivos soltos do tipo `exercicio1.c` ou `teste.c`. 

Para evitar essa bagunça, criei este **Hub de Algoritmos**. A ideia foi unificar os principais conceitos de lógica e estruturas de dados que aprendi na faculdade em um único programa modular. Tudo funciona dentro de um painel interativo direto no terminal do Windows, deixando o portfólio limpo e muito mais legal de mexer.

---

## 🚀 O que ele faz de verdade?

Aqui está o resumo do que você encontra rodando o menu do painel:

* **Menu Inteligente:** Um sistema de navegação estruturado em loop que não deixa o programa fechar sozinho após executar uma tarefa.
* **Módulo de Recursividade:** Testes práticos calculando Fatorial e um sistema de Par ou Ímpar baseado em funções que chamam uma à outra.
* **Manipulação de Matrizes:** Algoritmos que pegam uma tabela inteira (matriz) e transformam em uma linha só (vetor), e depois fazem o caminho inverso em tempo de execução.
* **Preenchimento Automático:** Para não ter que ficar digitando 20 números toda hora, o sistema usa o relógio do computador para gerar dados aleatórios para os testes.
* **Validador de Palíndromos:** Um leitor de texto que ignora espaços e quebras de linha para checar se uma palavra ou frase se lê igual de trás para frente.
* **Currículo no Terminal:** Um comando que limpa a tela do console e renderiza o meu currículo profissional formatado na hora.

## 📂 Estrutura do Fluxo

```text
 HUB_ALGORITMOS.EXE (Console)
 ├── [1] Algoritmos e Recursividade
 │    ├── Fatorial (Recursão Direta)
 │    └── Par ou Ímpar (Recursão Mútua)
 ├── [2] Manipulação de Matrizes
 │    ├── Linearizar (Matriz para Vetor)
 │    └── Reconstruir (Vetor para Matriz)
 ├── [3] Processamento de Strings
 │    └── Validador de Palíndromos
 └── [4] Terminal CV (Meu Currículo na Tela)
